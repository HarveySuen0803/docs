# Restful URL

| URL             | description           |     |
| :-------------- | :-------------------- | --- |
| GET /users      | select all user       |     |
| GET /users/1    | select user with id 1 |     |
| POST /users     | insert user           |     |
| PUT /users/1    | update user with id 1 |     |
| DELETE /users/1 | delete user with id 1 |     |
